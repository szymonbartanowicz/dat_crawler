## Instalacja
1. ``
git clone https://github.com/szymonbartanowicz/dat_crawler.git
``
2. ``
cd dat_crawler  
``
3. ``
pip install requirements.txt
``
4. ``
bash script.sh
``

W katalogu data zostaną utworzone 4 pliki .jsonl, po jednym na domenę.

## Statystki:
| Domena        | Czas    | Strony |
|---------------|---------|--------|
| dat           | ~5min   | ~300   |
| career_dat    | ~0.5min | ~25    |
| barometer_dat |         |        |
| top100_dat    |         |        |
